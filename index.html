What to look for in firewall settings

The goal of changing firewall settings is to close potentually dangerous ports you do not need open.

To change port configuration

  • Open Application Search
  • Search for "Windows Firewall With Advanced Security" app and launch it
  • Check inbound and outbound rules you don't need (like FTP traffic)
  • Disable those rules by right clicking them and selecting that option

How to identify suspicious files

Suspicious files can be hidden and obfuscated in order to stop the user from interfering with them

These tactics include:

  • Labeling files as "hidden" so they do not show in the file explorer by default
  • Using multiple file extensions to make powerful file types look like less powerful ones
  • Hiding files in obscure locations with obscure names.

These strategies can be midigated by showing file extensions and hidden files

Where do deleted files go?

It turns out, nowhere. Files are stored in sectors on a drive. When you want to access a file, the computer looks up the sector it is located in and then gets the file from that location.

when you delete a file, instead of wiping the sectors of the drive that the file is stored in, the computer chooses to save resources and time by just marking those sectors as "free to write to" as they will be eventaully overritten when that space is filled by new files.

Remember that without the use of third party programs, nothing is really deleted when you press that button

How do I know if bad stuff is running on my computer?

The Task Manager application shows all programs currently running on your computer. If there are programs that you are suspicious of, there are a few ways to check if they are legitiment

  • Check the program's location of origin in the file manager
  • Check the user who ran the program
  • Check the programs digital certificate in it's properties

If you do find a suspicious process, make sure to end the entire task tree as parent tasks can just relaunch malicious programs.

Deleting the files from the process origin is also good idea.
